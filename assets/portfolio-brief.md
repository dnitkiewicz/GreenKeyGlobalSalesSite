# Portfolio rollout angle for Green Key Global

This brief is intended to help shape the executive narrative around Green Key Global for a VP of operations at a hotel franchise management company.

## Core message
- The case is not only about sustainability. It is about operating evidence, commercial readiness, and portfolio discipline.
- The program gives leadership a credible way to show how hotels operate, without requiring a major capital project.
- It can support corporate RFP conversations, improve operating habits, and give owners a clearer view of performance and audit readiness.

## What executives need to hear
- Proof that the program can be implemented efficiently across multiple properties.
- A clear explanation of how the program affects revenue, cost, risk, and operating consistency.
- A practical path from pilot to portfolio scale.
- A cost structure that is understandable and manageable for hotel teams.

## Framing to use
- "We certify operations, not construction."
- "Certification is a proof point for how a property operates."
- "The program supports both the operating story and the commercial story."
- "A portfolio can move from pilot to standard with the same framework and a repeatable rollout plan."
