The 90-day path

Make the first move quickly without overcomplicating the process

The path is designed to fit inside one quarter with a small team and a clear owner. We never ask a property to be something it is not. We ask it to run well, prove it, and be recognized for it.

Start the Conversation See common objections

Days 1 to 30, foundation

Assign ownership

Name the accountable senior officer and form the Green Team with a written mandate and quarterly meetings.

Gather records

Request 12 months of utility invoices and collect the templates for policy, action plan, purchasing policy, and Green Team mandate.

Build the basic operating framework

Set the tone for the work that will soon become visible to the audit team.

Days 31 to 60, close the gaps

Housekeeping routines

Launch the linen and towel reuse program and reinforce the trained behaviors in housekeeping checklists.

Engineering discipline

Put the preventative maintenance program in writing and document fixture coverage and monitoring logs.

Procurement habits

Finalize and sign the purchasing policy and strengthen everyday purchasing practice.

Days 61 to 90, assess and request audit

Complete the self-assessment using the Core Criteria filter first, upload every required document, run a cross-department consistency check, and then request the audit through the Members Area.
