# Green Key Global Eco-Rating Summary Site

This workspace now contains a simple static website for sharing a summary of Green Key Global and the Eco-Rating certification roadmap.

## Files
- index.html — landing page for the site

## Deploy to GitHub Pages
1. Create a GitHub repository and push these files to the repository root.
2. In GitHub, open Settings → Pages.
3. Under Build and deployment, choose GitHub Actions.
4. Commit the workflow file in [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml).
5. Push to the main branch and GitHub Pages will publish the site automatically.

## Notes
The page is intentionally lightweight and static so it can be deployed without a build step.
